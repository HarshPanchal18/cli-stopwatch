# cli-stopwatch
## Stopwatch for >_ `TERMINAL`

----

* Method 1 (By Parameter)
> To Start the Counting
>> ./stopwatch.sh --start

> To Track the Counting
>> ./stopwatch.sh --prog

> To Stop the Counting
>> ./stopwatch.sh --stop

----

* Method 2 (By Individuals)
> To Start the Counting
>> ./stopwstart.sh

> To Track the Counting
>> ./stopwprogress.sh

> To Stop the Counting
>> ./stopwstop.sh

----
